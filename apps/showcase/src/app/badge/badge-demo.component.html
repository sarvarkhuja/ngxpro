Badge

Compact label for status, category, or metadata. Supports solid and dot variants with the full Tailwind color palette.

Solid

Default variant with tinted color background.

@for (c of sampleColors; track c) { {{ sampleLabels[c] }} }

Dot

Border style with a colored dot indicator.

@for (c of sampleColors; track c) { {{ sampleLabels[c] }} }

Sizes

Three sizes: sm, md, lg.

@for (s of sizes; track s) { {{ capitalize(s) }} }

Colors

All 17 Tailwind palette colors in both variants.

@for (c of allColors; track c) { {{ capitalize(c) }} }
@for (c of allColors; track c) { {{ capitalize(c) }} }

Directive usage

The nxpBadge directive can be applied to any inline element.
