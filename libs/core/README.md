# @ngxpro/core

Foundation services, pipes, and tokens for the [ngxpro](https://github.com/SarvarkhujaMurodov/ngxpro) Angular UI library.

Includes: `ThemeService`, `BreakpointService`, `FormatService`, plus `amount`, `currency`, `number`, and `relative-time` pipes.

## Install

```bash
npm install @ngxpro/core --legacy-peer-deps
```

## Peer dependencies

- `@angular/common` ^21.1
- `@angular/core` ^21.1
- `@ngxpro/cdk` (any matching workspace version)

## Usage

```ts
import { ThemeService, BreakpointService } from '@ngxpro/core';
```
