Avatar

User avatars with image, initials fallback, sizes, shapes, badge, labels, and outlines.

Initials fallback

When no src is set or the image fails to load, initials are derived from alt.

With image

Provide a src for a photo. A broken URL automatically falls back to initials.

With badge

Avatar stack

Use nxp-avatar-stack with direction="start" or "end" to overlap avatars into a group.

Labeled avatar

Wrap an avatar in nxp-avatar-labeled with a label to show a name or caption below it. Long labels are truncated.

Auto color

Use the nxpAutoColor pipe to generate a deterministic background color from any string (name, ID, email). The same input always produces the same color.

@for (name of autoColorNames; track name) { }
