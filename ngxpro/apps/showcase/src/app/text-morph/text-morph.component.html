Smooth FLIP-based text animations when content changes. Available as a directive or component.

Usage

Directive (recommended)

<h1 ngxproTextMorph [text]="title()" [duration]="400"></h1>

Component

<ngxpro-text-morph [text]="label()" class="text-xl" />

App-wide defaults

providers: [provideTextMorphOptions({{ '{' }} duration: 500, locale: 'ja' {{ '}' }})]
