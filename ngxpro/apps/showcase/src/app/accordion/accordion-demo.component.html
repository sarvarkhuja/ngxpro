What is nxp?

Production-ready Angular UI library
nxp is a production-ready Angular UI library built with Taiga architecture patterns and Tremor Tailwind styling. It provides 300+ components and blocks for building modern web applications.

Key Features

Taiga + Tremor patterns
  • Taiga UI architecture patterns
  • Tremor Tailwind styling
  • Light/Dark mode support
  • Fully responsive and accessible
  • 300+ components and blocks
Installation npm or pnpm
